# Objeto de Aprendizagem – TDD aplicado à UNIFACE

Este repositório contém um **Objeto de Aprendizagem** desenvolvido como parte da disciplina do **Mestrado em Ciência da Computação**.

O objetivo do projeto é apoiar o ensino prático da abordagem **Test-Driven Development (TDD)** aplicada ao desenvolvimento na plataforma **UNIFACE**, auxiliando estudantes e desenvolvedores a compreenderem como aplicar testes durante o processo de desenvolvimento.

---

# Objetivo do Projeto

O objeto de aprendizagem tem como finalidade:

- Apresentar os conceitos fundamentais de **TDD**
- Demonstrar como aplicar **TDD na plataforma UNIFACE**
- Auxiliar estudantes na compreensão do fluxo **Red → Green → Refactor**
- Proporcionar material visual e conceitual para estudo
